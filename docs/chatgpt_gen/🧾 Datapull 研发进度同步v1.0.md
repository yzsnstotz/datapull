## 二、文件使用说明

| 字段类型    | 说明                                                                     |

| 📅 周期字段 | 可按周、阶段或版本填写（例如 “2025W02”、“v1.0.1 测试前核查”）                               |
| 🧩 模块字段 | 按照项目架构（Crawler / Extractor / Chunker / Ingester / Service / Utils）逐项更新 |
| 📊 状态字段 | 使用统一 emoji 状态标识（✅ 完成，🟡 进行中，🔵 待开始，🔴 阻塞）                              |
| ⚙️ 任务字段 | 对应核对清单任务编号（例如：CR-01, EX-03）                                            |
| 🧭 审查字段 | 由 PM / Tech Lead 审阅后签署                                                 |
